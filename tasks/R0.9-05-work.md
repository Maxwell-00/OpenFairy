## Ctrl+C and cleanup state machine

1. CLI converts either `SIGINT` or `SIGTERM` to an abort signal.
2. Reused mode exits the launcher only and sends no process signal.
3. Owned mode sends the fixed IPC `fairy.launcher.shutdown` message.
4. The existing launcher forwards graceful `SIGTERM` to its gateway child.
5. Dev waits `5,000 ms`; if needed, Windows runs fixed `taskkill.exe /pid <owned-pid> /t /f` (Unix uses `SIGKILL`).
6. Dev waits for exit and polls until the selected loopback port is free.
