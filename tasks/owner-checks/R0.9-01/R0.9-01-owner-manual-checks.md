## 目标

完成两项检查：

1. **一条真实 MiMo pay-as-you-go ASR**
   - 普通 `sk-*` key
   - 一个 WAV/MP3
   - 1 worker / 1 provider request / 1 ASR final / 1 turn / 1 model request
2. **一条 secret/local-only denial**
   - 不得 spawn worker
   - 不得 staging
   - 不得连接 MiMo
   - 不得产生 final、turn 或 model request

MiMo 配置必须严格保持 `personal + [region-restricted, global-ok] + [cn]`，认证只使用 `api-key`。

```powershell
        throw "audio data URL leaked into $Path"
    }

    if ($Text -match '[A-Za-z0-9+/]{120,}={0,2}') {
        throw "long base64-like payload leaked into $Path"
    }

    if ($Text -match 'asr-input\.bin') {
        throw "worker input token leaked into $Path"
    }

    if ($Text -match 'fairy-mimo-asr-') {
        throw "temporary ASR root leaked into $Path"
    }

    if ($Text.IndexOf(
        $env:TEMP,
        [StringComparison]::OrdinalIgnoreCase
    ) -ge 0) {
        throw "temporary absolute path leaked into $Path"
    }
}
```
