## 4. Terminal C: start a deterministic local text model

Create the temporary script in Terminal B:

```powershell
@'
import { writeFileSync } from "node:fs";
import { MockOpenAIChatServer } from "../packages/testing/src/mock-openai.ts";

async function main(): Promise<void> {
  const outputPath = process.argv[2];

  if (!outputPath) {
    throw new Error("output path is required");
  }

  const server = await MockOpenAIChatServer.start({
    text: ["R0.9-05 deterministic owner model response."],
  });

  writeFileSync(outputPath, server.url, "utf8");
  console.log("mock model ready");

  const stop = async (): Promise<void> => {
    await server.stop();
    process.exit(0);
  };

  process.on("SIGINT", () => void stop());
  process.on("SIGTERM", () => void stop());

  await new Promise(() => undefined);
}

void main().catch((error: unknown) => {
  console.error(error);
  process.exitCode = 1;
});
'@ |
    Set-Content -Encoding utf8 $MockProgram
```

Because the script imports from the repository, copy it into repository `tmp/` instead of the Windows temp directory:

```powershell
New-Item -ItemType Directory -Force 'tmp' |
```

## 5. Terminal B: create the owner config

Wait for the URL:

```powershell
while (-not (Test-Path -LiteralPath $MockUrlFile)) {
    Start-Sleep -Milliseconds 100
}

$MockModelUrl =
    (Get-Content -Raw $MockUrlFile).Trim()
```

Create the config:

```powershell
$Example =
    Get-Content -Raw 'examples/fairy.v0.9.yaml'

$OwnerConfig = $Example.
    Replace(
        'https://example.invalid/v1',
        $MockModelUrl
    ).
    Replace(
        'replace-with-openai-compatible-model',
        'deterministic-owner-mock'
    ).
    Replace(
        'replace-with-minimax-voice-id',
        'male-qn-qingse'
    )

$OwnerConfig |
    Set-Content -Encoding utf8 $ConfigPath
```

Use a synthetic local model credential because the local mock does not call a public provider:

```powershell
$env:MAIN_MODEL_API_KEY =
    'synthetic-local-mock-key'
```

Generate a random gateway token:

```powershell
$env:GATEWAY_TOKEN =
    [Convert]::ToHexString(
        [Security.Cryptography.RandomNumberGenerator]::
            GetBytes(32)
    ).ToLowerInvariant()

$env:FAIRY_GATEWAY_TOKEN =
    $env:GATEWAY_TOKEN
```

Load real MiMo/MiniMax credentials through your existing secure method:

```text
MIMO_ASR_PAYGO
MINIMAX_T2A_TOKEN
```

Do not print their values.

Check only presence/class:

```powershell
[ordered]@{
    main_model_present =
        -not [string]::IsNullOrWhiteSpace(
            $env:MAIN_MODEL_API_KEY
        )
    gateway_present =
        -not [string]::IsNullOrWhiteSpace(
            $env:GATEWAY_TOKEN
        )
    mimo_present =
        -not [string]::IsNullOrWhiteSpace(
            $env:MIMO_ASR_PAYGO
        )
    mimo_paygo_class =
        (
            -not [string]::IsNullOrWhiteSpace(
                $env:MIMO_ASR_PAYGO
            ) -and
            $env:MIMO_ASR_PAYGO.StartsWith('sk-')
        )
    minimax_present =
        -not [string]::IsNullOrWhiteSpace(
            $env:MINIMAX_T2A_TOKEN
        )
} |
    Format-List
```

All values must be `True`.
